## AI小助手
>这是一个基于Java和ChatGPT开发的AI小助手项目，它可以完成用户的自然语言处理和智能对话交互。
用户可以通过与AI小助手的文字对话来获取帮助、解决问题等，AI小助手使用ChatGPT模型进行智能回答。


## 运行条件
> *  Redis
> *  MySQL数据库
